En el siguiente Objeto Virtual de Información (OVI) que pertenece al curso académico Diseño web encontrará material relacionado con la Unidad 1 denominada "Construccion de un sitio web".

La revisión de este OVI le permitirá comprender la construccion de un sitio web utilizando herramientas como HTML , CSS y JavaScript, con el cual podrá aportar de manera pertinente al desarrollo del Trabajo colaborativo del curso.
