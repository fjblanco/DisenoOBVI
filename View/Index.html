(Objeto Virtual de Información)

Para el curso de Diseños de Sitios Web

En el cual encontrara información acerca de como maquetar y estructurar un sitio web desde cero, agregando estilos con CSS y CSS 3, funciones JAVASCRIPT y JQUERY para animaciones del sitio.

1. Crear un sitio web
Para empezar a crear un sitio web debemos:

1. Crear un archivo .html
2. Abrir nuestro archivo en cualquier editor de texto.
2. Maquetar sitio web
Despues de crear el archivo .html seguimos con el siguiente paso que es maquetar el sitio web y para esto debemos agregar dentro del archivo antes creado las siguientes etiquetas como se muestra en la imagen.

Ver imagen
3. Ejecutar o probar sitio web
por ultimo debemos probar nuestro sitio creado , para probarlo solo debemos guardar y abrilo en un navegado ( el de su preferencia)
